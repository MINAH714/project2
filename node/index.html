우리의 AI 공감 서비스란?

우리는 최신 인공지능 기술을 활용하여 사용자의 대화 패턴과 감정 상태를 분석하고, 이에 기반한 공감적인 대화문을 생성합니다. 이는 사용자가 자신의 감정을 더 잘 이해하고, 건강한 소통 방식을 연습하는 데 도움을 줍니다. 개인의 성장과 관계 개선을 위한 강력한 도구입니다.

개인화된 대화

사용자의 특성(나이, 성별, 상황)에 맞춰 최적화된 대화 스크립트를 제공합니다.

정교한 감정 분석

대화 내 감정 변화를 다각도로 분석하여 심층적인 인사이트를 제공합니다.

주요 기능

AI 챗봇

다양한 상황과 감정을 반영한 AI챗봇과의 대화를 나눌수 있습니다

감정 분석 리포트

생성된 대화 내 감정의 분포와 변화를 상세한 리포트로 제공합니다.

S3 데이터 저장

생성된 대화 데이터를 안전하게 AWS S3 버킷에 저장할 수 있습니다.
